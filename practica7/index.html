Automatización Electrónica Circuitos

- Resumen -

Se realizó una investigación sobre los componentes utilizados en los circuitos de control industrial, para lo cual se consultaron diversas fuentes bibliográficas y se realizaron prácticas en el laboratorio. Se utilizó el software Fluidsim para realizar la simulación del circuito de control industrial. Se utilizó la versión de diagrama de escalera y relevadores. Se utilizó un PLC S71200-1215C para la construcción del circuito de control industrial. Se utilizó el software Tia Portal para la programación del PLC.

La principal diferencia entre los circuitos es que el circuito simulado en Fluidsim utiliza relevadores, mientras que el circuito construido con un PLC utiliza contactos programables. Otra diferencia es que el circuito simulado en Fluidsim es más sencillo de implementar, ya que no requiere de un hardware específico.

- Introducción -

Los PLC tienen las siguientes características:

● Son dispositivos programables que pueden ser utilizados para controlar procesos industriales.

● Son más flexibles que los sistemas de control basados en relevadores.

● Pueden ser configurados para adaptarse a diferentes aplicaciones.

● Son más fáciles de mantener que los sistemas de control basados en relevadores.

- Resultados y Conclusiones-

La práctica permitió conocer y comprender el funcionamiento de un PLC.

Se identificaron las principales diferencias entre los circuitos simulados y construidos con un PLC.

Se comprendieron las características de utilizar un PLC.

Los PLC son dispositivos programables que permiten una mayor versatilidad en el control de procesos.Son más flexibles que los sistemas de control basados en relevadores, ya que su lógica de control se puede modificar fácilmente mediante programación.Los PLC pueden configurarse para adaptarse a diferentes aplicaciones, lo que los hace ideales para una amplia gama de aplicaciones industriales.Son más fáciles de mantener que los sistemas de control basados en relevadores, ya que los cambios en la lógica de control se pueden realizar mediante software en lugar de realizar cambios físicos en el cableado.
